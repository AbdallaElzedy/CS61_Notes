strtol() Base Conversion Reference

Base String Example Result Notes
0 "0xFF" 255 Auto-detect: Detects hex from 0x prefix
0 "077" 63 Auto-detect: Detects octal from leading 0
0 "123" 123 Auto-detect: No prefix means decimal
2 "1010" 10 Binary: Only 0 and 1 allowed
2 "11111111" 255 Binary: Eight 1s equals 255
8 "777" 511 Octal: Digits 0-7 only
8 "100" 64 Octal: 1×8² + 0×8¹ + 0×8⁰
10 "12345" 12345 Decimal: Standard base-10 (most common)
10 "-999" -999 Decimal: Negative numbers work too
16 "FF" 255 Hexadecimal: F = 15, so 15×16 + 15 = 255
16 "1A2B" 6699 Hexadecimal: Digits 0-9, A-F (case-insensitive)
16 "CAFE" 51966 Hexadecimal: Common for colors and memory addresses
36 "HELLO" 29234652 Base-36: Uses 0-9 and A-Z (max base)
36 "Z" 35 Base-36: Z is the highest digit (35)
// Usage Examples:

long bin = strtol("1010", NULL, 2); // Binary: 10
long oct = strtol("777", NULL, 8); // Octal: 511
long dec = strtol("100", NULL, 10); // Decimal: 100
long hex = strtol("FF", NULL, 16); // Hex: 255
long auto1 = strtol("0xFF", NULL, 0); // Auto-detect hex: 255
long auto2 = strtol("077", NULL, 0); // Auto-detect octal: 63
long max = strtol("HELLO", NULL, 36); // Base-36: 29234652
